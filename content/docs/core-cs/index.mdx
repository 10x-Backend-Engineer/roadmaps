### 📝 How to use this list

1.  **Pick a level:** Start with "Beginner" if you are new to DSA.
2.  **Choose a project:** Pick one that interests you.
3.  **Identify the concepts:** Look at the parentheses (e.g., `Graph + Dijkstra`) to know what you need to study.
4.  **Build it:** Code the project from scratch without looking at tutorials first!
